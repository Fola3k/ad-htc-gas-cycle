# AD-HTC Fuel-Enhanced Power Gas Cycle

**Energhx Research Group · Faculty of Engineering, University of Lagos**

An interactive thermodynamic analysis tool for the Anaerobic Digestion–Hydrothermal Carbonisation (AD-HTC) Fuel-Enhanced Gas Power Cycle.

## Features
- Interactive animated process schematic with embedded parameter badges
- h–s diagram for the HTC Steam (Rankine) Cycle
- T–Ḣ diagram for the Brayton Gas Power Cycle (rigorous CoolProp evaluations)
- Real-time state property tables for all cycle state points
- Biomass mass flow routing and energy balance summaries

## Requirements
- Python 3.10 or higher
- See `requirements.txt` for dependencies

## How to Run

### 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/ad-htc-gas-cycle.git
cd ad-htc-gas-cycle

### 2. (Recommended) Create a virtual environment
python -m venv venv

# Activate on Windows:
venv\Scripts\activate

# Activate on Mac/Linux:
source venv/bin/activate

### 3. Install dependencies
pip install -r requirements.txt

### 4. Run the app
streamlit run ad_htc_app.py

The app will open automatically in your browser at http://localhost:8501
